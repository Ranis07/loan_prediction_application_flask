This is a loan predicting web application that predicts the future eligibility of the loan. This product can help financial institutions like bank, sahakari sanstha, etc. It helps their customer know about loan eligibility via online which makes work much easier, faster and convenient. This product is developed by using supervised machine learning with train and test dataset used from Kaggle. Later, Logistic Regression model was implemented since it gave high accuracy than the rest. And for web application, flask framework was used.
